# ICG Project #1: Shading

### Requirements

- [x] Implement 3 shading methods
  - [x] Flat shading
  - [x] Gouraud shading
  - [x] Phong shading
- [ ] Enable 4 transformations
  - [x] Translation
  - [x] Scale
  - [x] Rotation
  - [ ] Shear
- [ ] Display at least 3 objects in a scene
- [ ] At least 3 light sources
