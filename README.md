# ICG Project #1: Shading

### Requirements

- [x] Implement 3 shading methods
  - [x] Flat shading
  - [x] Gouraud shading
  - [x] Phong shading
- [x] Enable 4 transformations
  - [x] Translation
  - [x] Scale
  - [x] Rotation
  - [x] Shear
- [x] Display at least 3 objects in a scene
- [x] At least 3 light sources
